# Agent Notes

- Base: main
- Head: feat/commissions
- Title: "feat: implement Commission system with currency"
- Body: describe what was built (commands, data storage, config)
- Use: gh pr create --base main --head feat/commissions --title "feat: implement Commission system with currency" --body "..."

Make sure gh is authenticated or use the API via curl with the GITHUB_TOKEN.
